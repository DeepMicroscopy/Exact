# Keyboard Shortcuts

## Annotation Viewer

| Key | Modifier | Function |
|---|:---:|---|
| `Del`, `x` | | Delete selected annotation |
| `Escape` | | Cancel drawing / deselect |
| `Enter` | | Confirm / save annotation |
| `Ctrl+Z` | | Undo last vertex (polygon) |
| `c` | | Toggle annotation mode (draw ↔ select) |
| `y` | | Toggle annotation visibility |
| `b` | | Push active annotation type to background |
| `Ctrl+F` | | Find a file in the image list |
| `Ctrl+A` | | Draw on top of existing annotation |
| `0–4` | | Change label of selected local annotation |
| `0–4` | `Shift` | Change label of selected global annotation |
| `q` | | Previous image |
| `q` | `Shift` | Previous frame (z-stack / volume) |
| `e` | | Next image |
| `e` | `Shift` | Next frame (z-stack / volume) |
| `r` | | Rotate image *(annotations not affected)* |
| `f` | | Flip image *(annotations not affected)* |
| '^' | | Continue screening |
| Arrow keys | | Move viewport |
| Mouse wheel | `Shift` | Adjust paint brush size |

### Polygon Edit Tools

These keys activate special edit tools when a polygon annotation is selected:

| Key | Tool | Function |
|---|---|---|
| `s` | Scissor | Subtract a drawn area from the polygon |
| `g` | Glue | Add a drawn area to the polygon |
| `d` | Knife | Draw a line to split the polygon into two |

## Screening Viewer

The screening viewer divides a WSI into a grid and lets you systematically scan every tile.

| Key | Function |
|---|---|
| `a` | Mark and move to left tile |
| `d` | Mark and move to right tile |
| `w` | Mark and move to up tile |
| `s` | Mark and move to down tile |
| `j` | Navigate to left tile (no marking) |
| `l` | Navigate to right tile (no marking) |
| `i` | Navigate to up tile (no marking) |
| `k` | Navigate to down tile (no marking) |
